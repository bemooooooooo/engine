Detailed Description

Encapsulates a window using GLFW for context and event handling.

This class manages the creation, updating, and destruction of a window. It also handles input events and provides a callback mechanism for event processing.

Key features:

  • Create and destroy a window with specified properties
  • Poll events and swap buffers each frame
  • Set event callback functions for handling input and window events
  • Query window dimensions and native handle
Note
Requires GLFW library to be linked with the project.
