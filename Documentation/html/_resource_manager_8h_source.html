11/**
12 * @class ResourceManager
13 * @brief Centralized manager for loading, storing, and accessing graphics resources.
14 *
15 * This class provides static methods to load, retrieve, and unload various graphics
16 * resources such as shaders, textures, and meshes. It maintains internal maps to
17 * ensure that resources are loaded only once and shared across the engine.
18 *
19 * Key features:
20 * - Load shaders from files or source code
21 * - Load textures from image files
22 * - Register and retrieve meshes
23 * - Automatic resource cleanup on shutdown
24 *
25 * @note All methods are static; no instances of ResourceManager should be created.
26 * @note Resources are identified by unique string names.
27 *
28 * @example
29 * @code
30 * auto shader = ResourceManager::LoadShader("basic", "vertex.glsl", "fragment.glsl");
31 * auto texture = ResourceManager::LoadTexture("brick", "brick_diffuse.png");
32 * auto mesh = ResourceManager::GetMesh("cube");
33 * @endcode
34 */
