5/**
6 * @class Texture
7 * @brief Represents a 2D texture resource in OpenGL.
8 *
9 * This class encapsulates the creation, management, and usage of 2D textures
10 * in OpenGL. It provides methods to load textures from image files, create
11 * empty textures, bind/unbind them for rendering, and set texture data.
12 *
13 * Key features:
14 * - Load textures from image files (e.g., PNG, JPEG)
15 * - Create empty textures with specified width and height
16 * - Bind and unbind textures to texture units
17 * - Set texture data dynamically
18 *
19 * @note Requires a valid OpenGL context to function properly.
20 * @note All OpenGL calls are made through this class; direct OpenGL calls should be avoided.
21 *
22 * @example
23 * @code
24 * // Create a texture from file
25 * Ref<Texture> texture = Texture::Create("path/to/texture.png");
26 *
27 * // Create an empty texture
28 * Ref<Texture> emptyTexture = Texture::Create(512, 512);
29 *
30 * // Bind the texture for use
31 * texture->Bind(0); // Bind to texture unit 0
32 *
33 * // Set texture data
34 * void* data = ...; // Pointer to pixel data
35 * texture->SetData(data, width * height * 4); // Assuming RGBA format
36 *
37 * // Unbind the texture
38 * texture->Unbind();
39 * @endcode
40 */
