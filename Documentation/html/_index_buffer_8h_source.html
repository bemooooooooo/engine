4/**
5 * @class IndexBuffer
6 * @brief Represents an OpenGL index buffer for indexed drawing.
7 *
8 * The IndexBuffer class encapsulates the creation, binding, and management
9 * of an OpenGL index buffer object (IBO). It allows for efficient rendering
10 * of complex geometries by reusing vertex data through indexed drawing.
11 *
12 * Key features:
13 * - Creation of index buffer from an array of indices
14 * - Binding and unbinding the index buffer for rendering
15 * - Retrieval of the number of indices stored
16 *
17 * Usage:
18 * - Create an IndexBuffer with a list of indices
19 * - Bind the buffer before issuing draw calls
20 * - Unbind when done to avoid unintended state changes
21 *
22 * @note This class assumes that an OpenGL context is active when used.
23 * @note The indices are expected to be of type u32 (unsigned 32-bit integers).
24 *
25 * @example
26 * @code
27 * u32 indices[] = {0, 1, 2, 2, 3, 0};
28 * IndexBuffer ibo(indices, 6);
29 *
30 * ibo.Bind();
31 * GraphicsContext::DrawElements(ibo.GetCount());
32 * ibo.Unbind();
33 * @endcode
34 */

36public:
37 /**
