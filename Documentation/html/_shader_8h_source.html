13 * @class Shader
14 * @brief Represents an OpenGL shader program composed of vertex and fragment shaders.
15 *
16 * This class encapsulates the creation, compilation, linking, and management of
17 * OpenGL shader programs. It provides methods to bind/unbind the shader and set
18 * uniform variables of various types (int, float, vectors, matrices).
19 *
20 * Key features:
21 * - Load shaders from source code strings or files
22 * - Compile and link shader programs
23 * - Set uniform variables with caching for performance
24 *
25 * @note Requires a valid OpenGL context to function properly.
26 * @note All OpenGL calls are made through this class; direct OpenGL calls should be avoided.
27 *
28 * @example
29 * @code
30 * std::string vertexSrc = "..."; // Vertex shader source code
31 * std::string fragmentSrc = "..."; // Fragment shader source code
32 * Ref<Shader> shader = Shader::Create(vertexSrc, fragmentSrc);
33 *
34 * shader->Bind();

38 * @endcode
