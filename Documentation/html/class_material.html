Detailed Description

Represents a material with properties and associated shader.

The Material class encapsulates the visual properties of a surface, including its shader and material properties such as color, metallicity, roughness, and texture maps. It provides methods to apply these properties to a shader for rendering.

Key features:

  • Holds a reference to a Shader used for rendering
  • Contains MaterialProperties defining appearance
  • Method to apply properties to the shader uniforms
