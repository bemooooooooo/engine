Detailed Description

Represents an OpenGL shader program composed of vertex and fragment shaders.

This class encapsulates the creation, compilation, linking, and management of OpenGL shader programs. It provides methods to bind/unbind the shader and set uniform variables of various types (int, float, vectors, matrices).

Key features:

  • Load shaders from source code strings or files
  • Compile and link shader programs
  • Set uniform variables with caching for performance
Note
Requires a valid OpenGL context to function properly.
All OpenGL calls are made through this class; direct OpenGL calls should be avoided.
