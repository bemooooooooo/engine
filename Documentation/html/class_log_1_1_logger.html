Detailed Description

Logger class for creating log messages with various severity levels.

The Logger class provides a stream-style interface for constructing log messages. It captures the log level, source file, and line number for each message. Messages are output to the console and optionally to a log file based on the configured log level.

Usage:

Log::Logger(Level::_Info, __FILE__, __LINE__) << "This is an info message.";
