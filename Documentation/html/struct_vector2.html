Detailed Description

A 2-dimensional vector structure for mathematics and graphics operations.

Vector2 represents a point or direction in 2D space using two floating-point components (x, y). It supports standard vector operations including addition, subtraction, scalar multiplication, normalization, and distance calculations.

Common uses:

  • 2D positions and velocities
  • Screen coordinates and UI positioning
  • Direction vectors for 2D movement
  • Texture coordinates (UV mapping)
  • 2D physics calculations
Note
All components are stored as f32 (single-precision floats) for performance.
Vector operations are defined as operator overloads for intuitive syntax.
