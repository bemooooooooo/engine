\doxysubsection{Detailed Description}
Encapsulates the properties and texture maps of a material. 

This structure holds the various properties that define the appearance of a material in the rendering engine, including color, metallicity, roughness, ambient occlusion, and associated texture maps.

Properties:\+
\begin{DoxyItemize}
\item albedo:\+ Base color of the material
\item metallic:\+ Metallic factor (0.\+0 = non-\/metal, 1.\+0 = metal)
\item roughness:\+ Surface roughness (0.\+0 = smooth, 1.\+0 = rough)
\item ambient\+Occlusion:\+ Ambient occlusion factor
\end{DoxyItemize}

\doxylink{class_texture}{Texture} Maps:\+
\begin{DoxyItemize}
\item albedo\+Map:\+ \doxylink{class_texture}{Texture} for base color
\item normal\+Map:\+ \doxylink{class_texture}{Texture} for surface normals
\item metallic\+Map:\+ \doxylink{class_texture}{Texture} for metallic values
\item roughness\+Map:\+ \doxylink{class_texture}{Texture} for roughness values
\item ambient\+Occlusion\+Map:\+ \doxylink{class_texture}{Texture} for ambient occlusion values
