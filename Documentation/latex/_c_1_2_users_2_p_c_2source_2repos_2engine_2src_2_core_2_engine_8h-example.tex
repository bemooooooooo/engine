Core engine class managing the main application loop and subsystems. The \doxylink{class_engine}{Engine} class is responsible for initializing and shutting down the engine, managing the main loop, processing input, updating game logic, and rendering. It serves as the central hub that coordinates various subsystems such as windowing, timing, and event handling.\+Key responsibilities:\+


\begin{DoxyItemize}
\item Initialize and shut down the engine and its subsystems
\item Manage the main application loop
\item Process input events
\item Update game logic with frame-\/rate independent timing
\item Render frames to the window
\item Provide access to timing information (delta time, FPS)
\item Handle window events and callbacks
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This is a base class meant to be extended by specific game implementations. Derived classes should override virtual methods to implement game-\/specific behavior.
\end{DoxyNote}
