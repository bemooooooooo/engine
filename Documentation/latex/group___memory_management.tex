\doxysubsection{Detailed Description}
Template aliases and factory functions for managing dynamic memory using C++11 smart pointers. These ensure proper resource management and prevent memory leaks by automating cleanup. Use \doxylink{group___memory_management_gaa2eeed5b5b4a0ea20bd3dae08409662d}{Scope} for exclusive ownership and \doxylink{group___memory_management_gadd5e90b302c31b74a46619f240214bcc}{Ref} for shared ownership scenarios.
