\doxysubsection{Detailed Description}
Encapsulates a window using GLFW for context and event handling. 

This class manages the creation, updating, and destruction of a window. It also handles input events and provides a callback mechanism for event processing.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Create and destroy a window with specified properties
\item Poll events and swap buffers each frame
\item Set event callback functions for handling input and window events
\item Query window dimensions and native handle
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Requires GLFW library to be linked with the project. 
\end{DoxyNote}
