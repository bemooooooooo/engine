\doxysubsection{Detailed Description}
A 2-\/dimensional vector structure for mathematics and graphics operations. 

\doxylink{struct_vector2}{Vector2} represents a point or direction in 2D space using two floating-\/point components (x, y). It supports standard vector operations including addition, subtraction, scalar multiplication, normalization, and distance calculations.

Common uses:\+
\begin{DoxyItemize}
\item 2D positions and velocities
\item Screen coordinates and UI positioning
\item Direction vectors for 2D movement
\item \doxylink{class_texture}{Texture} coordinates (UV mapping)
\item 2D physics calculations
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All components are stored as \doxylink{group___base_types_gad34d88453d37b65a09797bad37f2f527}{f32} (single-\/precision floats) for performance. 

Vector operations are defined as operator overloads for intuitive syntax. 
\end{DoxyNote}
