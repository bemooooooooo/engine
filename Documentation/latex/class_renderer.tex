\doxysubsection{Detailed Description}
Core rendering system for managing draw calls and scene rendering. 

This class provides static methods for initializing the renderer, beginning and ending scenes, submitting draw calls, and handling window resize events. It manages the rendering state and coordinates with shaders and vertex arrays to render 3D objects to the screen.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item \doxylink{class_scene}{Scene} management (begin/\+end)
\item Draw call submission
\item Clear color and buffer clearing
\item \doxylink{class_window}{Window} resize handling
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All methods are static; no instances of \doxylink{class_renderer}{Renderer} should be created. 

Must be initialized via Init() before use. 
\end{DoxyNote}
