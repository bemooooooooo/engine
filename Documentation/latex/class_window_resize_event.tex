\doxysubsection{Detailed Description}
\doxylink{class_event}{Event} triggered when the window is resized. 

This event is dispatched whenever the window size changes, providing the new width and height. It allows the application to adjust rendering parameters or UI layouts accordingly. \begin{Desc}
