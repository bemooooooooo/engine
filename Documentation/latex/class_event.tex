\doxysubsection{Detailed Description}
Base class for all events in the event system. 

The \doxylink{class_event}{Event} class serves as the base for all specific event types in the engine\textquotesingle{}s event handling system. It provides common functionality such as type identification and a handled flag to indicate whether the event has been processed.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Type identification via \doxylink{class_event_a092ad068c77c897d25aec42563fb60ec}{Get\+Type()} and \doxylink{class_event_a759eeb1cb569cbea0c0a9707696adf6e}{Is\+Type$<$\+T$>$()}
\item Handled flag to prevent further processing of already handled events
\item \doxylink{class_event_a17303a8089a66eece86b41cdea845f55}{To\+String()} method for easy logging and debugging
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All specific event types should inherit from this class and implement their own type identification methods. 
\end{DoxyNote}
