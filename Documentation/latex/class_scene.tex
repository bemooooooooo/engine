\doxysubsection{Detailed Description}
Represents a 3D scene containing multiple scene objects and a camera. 

The \doxylink{class_scene}{Scene} class manages a collection of \doxylink{struct_scene_object}{Scene\+Object} instances, allowing for adding, removing, updating, and rendering them. It also holds a \doxylink{class_camera}{Camera} to define the viewpoint for rendering the scene.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Add and remove Scene\+Objects by name
\item Find Scene\+Objects by name
\item Update all objects each frame, invoking their on\+Update callbacks
\item Render all objects using a specified shader
\item Manage a \doxylink{class_camera}{Camera} for the scene
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The \doxylink{class_scene}{Scene} does not handle physics or collision; it focuses on object management and rendering. 
\end{DoxyNote}
