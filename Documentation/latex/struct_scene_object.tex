\doxysubsection{Detailed Description}
Represents an object within a 3D scene, containing a mesh and transform. 

\doxylink{struct_scene_object}{Scene\+Object} encapsulates a 3D mesh along with its transformation (position, rotation, scale) in the scene. It also supports an optional update callback for per-\/frame logic.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Holds a reference to a \doxylink{class_mesh}{Mesh} for rendering
\item Contains a \doxylink{class_transform}{Transform} for spatial positioning
\item Supports an on\+Update callback for custom behavior each frame
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The on\+Update function, if set, is called during the scene\textquotesingle{}s Update() method with the delta time parameter. 
\end{DoxyNote}
