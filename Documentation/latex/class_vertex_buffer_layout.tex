\doxysubsection{Detailed Description}
Defines the layout of vertex attributes in a vertex buffer. 

Manages a list of \doxylink{struct_vertex_buffer_element}{Vertex\+Buffer\+Element} instances and calculates the stride (total size) of the vertex structure.
