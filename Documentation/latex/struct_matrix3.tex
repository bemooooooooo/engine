\doxysubsection{Detailed Description}
A 3x3 matrix structure for mathematics and graphics operations. 

\doxylink{struct_matrix3}{Matrix3} represents a 3x3 matrix used for various transformations in 2D and 3D space, including rotation, scaling, and shearing. It supports standard matrix operations such as addition, subtraction, multiplication, transposition, and inversion. Common uses:\+
\begin{DoxyItemize}
\item 2D transformations (rotation, scaling, shearing)
\item Normal matrix calculations in 3D graphics
\item Linear transformations in 3D space
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All components are stored as \doxylink{group___base_types_gad34d88453d37b65a09797bad37f2f527}{f32} (single-\/precision floats) for performance. 

Matrix operations are defined as operator overloads for intuitive syntax. 
\end{DoxyNote}
