\doxysubsection{Detailed Description}
High-\/precision timer for measuring elapsed time and frame delta times. 

This class provides a flexible timer that can measure both total elapsed time since creation/\+reset and frame-\/to-\/frame delta times. It uses the system\textquotesingle{}s high-\/resolution clock for accurate timing measurements needed in game loops.

The timer is useful for:\+


\begin{DoxyItemize}
\item Frame timing and delta time calculation
\item Performance profiling and benchmarking
\item Game logic that depends on elapsed time
\item Animations and transitions
\item \doxylink{class_input}{Input} response timing
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This class is not thread-\/safe. Use separate \doxylink{class_time_1_1_timer}{Timer} instances for different threads. 
\end{DoxyNote}
