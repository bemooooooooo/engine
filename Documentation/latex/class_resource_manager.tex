\doxysubsection{Detailed Description}
Centralized manager for loading, storing, and accessing graphics resources. 

This class provides static methods to load, retrieve, and unload various graphics resources such as shaders, textures, and meshes. It maintains internal maps to ensure that resources are loaded only once and shared across the engine.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Load shaders from files or source code
\item Load textures from image files
\item Register and retrieve meshes
\item Automatic resource cleanup on shutdown
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All methods are static; no instances of \doxylink{class_resource_manager}{Resource\+Manager} should be created. 

Resources are identified by unique string names. 
\end{DoxyNote}
