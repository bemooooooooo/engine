\doxysubsection{Detailed Description}
A 3D camera class supporting perspective and orthographic projections. 

The \doxylink{class_camera}{Camera} class encapsulates the properties and behaviors of a virtual camera in a 3D scene. It supports both perspective and orthographic projection modes, allowing for flexible rendering setups. The camera maintains its position, orientation, and projection parameters, and provides methods to manipulate its transform and retrieve view/\+projection matrices.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Switchable between perspective and orthographic projections
\item Position and rotation management using \doxylink{struct_vector3}{Vector3} and \doxylink{struct_quaternion}{Quaternion}
\item View and projection matrix generation
\item Utility methods for moving, rotating, and looking at targets
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The camera uses a right-\/handed coordinate system. 

The up vector is assumed to be (0, 1, 0) unless specified otherwise. 
\end{DoxyNote}
