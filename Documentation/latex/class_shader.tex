\doxysubsection{Detailed Description}
Represents an Open\+GL shader program composed of vertex and fragment shaders. 

This class encapsulates the creation, compilation, linking, and management of Open\+GL shader programs. It provides methods to bind/\+unbind the shader and set uniform variables of various types (int, float, vectors, matrices).

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Load shaders from source code strings or files
\item Compile and link shader programs
\item Set uniform variables with caching for performance
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Requires a valid Open\+GL context to function properly. 

All Open\+GL calls are made through this class; direct Open\+GL calls should be avoided. 
\end{DoxyNote}
