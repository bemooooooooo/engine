\doxysubsection{Detailed Description}
Represents a 2D texture resource in Open\+GL. 

This class encapsulates the creation, management, and usage of 2D textures in Open\+GL. It provides methods to load textures from image files, create empty textures, bind/\+unbind them for rendering, and set texture data.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Load textures from image files (e.\+g., PNG, JPEG)
\item Create empty textures with specified width and height
\item Bind and unbind textures to texture units
\item Set texture data dynamically
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Requires a valid Open\+GL context to function properly. 

All Open\+GL calls are made through this class; direct Open\+GL calls should be avoided. 
\end{DoxyNote}
