\doxysubsection{Detailed Description}
Represents a material with properties and associated shader. 

The \doxylink{class_material}{Material} class encapsulates the visual properties of a surface, including its shader and material properties such as color, metallicity, roughness, and texture maps. It provides methods to apply these properties to a shader for rendering.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Holds a reference to a \doxylink{class_shader}{Shader} used for rendering
\item Contains \doxylink{struct_material_properties}{Material\+Properties} defining appearance
\item Method to apply properties to the shader uniforms
