\doxysubsection{Detailed Description}
Represents an Open\+GL index buffer for indexed drawing. 

The \doxylink{class_index_buffer}{Index\+Buffer} class encapsulates the creation, binding, and management of an Open\+GL index buffer object (IBO). It allows for efficient rendering of complex geometries by reusing vertex data through indexed drawing.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Creation of index buffer from an array of indices
\item Binding and unbinding the index buffer for rendering
\item Retrieval of the number of indices stored
\end{DoxyItemize}

Usage:\+
\begin{DoxyItemize}
\item Create an \doxylink{class_index_buffer}{Index\+Buffer} with a list of indices
\item Bind the buffer before issuing draw calls
\item Unbind when done to avoid unintended state changes
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This class assumes that an Open\+GL context is active when used. 

The indices are expected to be of type \doxylink{group___base_types_ga70c1fcc1a4e96af8f6580f708657cd41}{u32} (unsigned 32-\/bit integers). 
\end{DoxyNote}
