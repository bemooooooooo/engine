\doxysubsection{Detailed Description}
Logging utilities and logger class for engine diagnostics. 

Provides a flexible logging system with multiple severity levels (Trace, Debug, Info, Warn, Error, Critical). Supports logging to console and optional file output. Designed for easy integration into the engine for runtime diagnostics, debugging, and error reporting.

Features:\+
\begin{DoxyItemize}
\item Stream-\/style logging with operator\texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} overloads
\item Configurable log levels to filter messages
\item Optional file logging with append/\+overwrite modes
\end{DoxyItemize}

Usage:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{ENGINE\_INFO\ <<\ \textcolor{stringliteral}{"{}Engine\ started\ successfully."{}};}
\DoxyCodeLine{ENGINE\_WARN\ <<\ \textcolor{stringliteral}{"{}Low\ memory\ warning."{}};}
\DoxyCodeLine{ENGINE\_ERROR\ <<\ \textcolor{stringliteral}{"{}Failed\ to\ load\ texture:\ "{}}\ <<\ texturePath;}

\end{DoxyCode}
