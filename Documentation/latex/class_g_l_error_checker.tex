\doxysubsection{Detailed Description}
Utility class for checking and logging Open\+GL errors. 

This class provides functionality to check for Open\+GL errors after Open\+GL function calls. It logs any detected errors using the engine\textquotesingle{}s logging system. The class can be used in a RAII manner to automatically check for errors when going out of scope.
