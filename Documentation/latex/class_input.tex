\doxysubsection{Detailed Description}
Static class for handling input from keyboard and mouse. 

Provides methods to query the current state of keys and mouse buttons, as well as mouse position and scroll delta. Also allows locking the cursor.

\begin{DoxyNote}{Note}
Must be updated each frame via Update() to track state changes. 
\end{DoxyNote}
