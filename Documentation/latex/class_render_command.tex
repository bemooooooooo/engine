\doxysubsection{Detailed Description}
Static wrapper class for issuing rendering commands. 

This class provides a simplified interface for common rendering operations by wrapping the \doxylink{class_graphics_context}{Graphics\+Context} static methods. It serves as a higher-\/level abstraction for issuing draw calls and managing rendering state.

All methods are static, allowing direct access without instantiation.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Viewport setup
\item Clear color and buffer clearing
\item Drawing operations (arrays and indexed)
\item Depth testing and blending control
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This class relies on \doxylink{class_graphics_context}{Graphics\+Context} for actual Open\+GL calls. 

Must be initialized via Init() before use. 
\end{DoxyNote}
