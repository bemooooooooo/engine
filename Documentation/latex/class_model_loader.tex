\doxysubsection{Detailed Description}
Utility class for loading 3D models and meshes from files. 

This class uses the Assimp library to load 3D model files in various formats (e.\+g., OBJ, FBX, STL) and converts them into \doxylink{class_mesh}{Mesh} objects usable by the engine. It handles parsing the model file, extracting vertex and index data, and creating \doxylink{class_mesh}{Mesh} instances with associated materials.

\doxylink{namespace_key}{Key} features:\+
\begin{DoxyItemize}
\item Load single meshes or entire models with multiple meshes
\item Support for common 3D file formats via Assimp
\item Automatic processing of vertices, indices, normals, and texture coordinates
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Requires Assimp library to be linked with the project. 

Currently does not handle animations or skeletal data. 
\end{DoxyNote}
